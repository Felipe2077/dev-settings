## Dev environment settings

### GIT

Execute o comando abaixo para habilitar o VSCODE como editor de texto do gitconfig:

    git config --global core.editor code

Em seguida execute:

    git config --global --edit

Cole o conteúdo do arquivo ".gitconfig" deste repositório e salve.

## Padrão de commits

Use o padrão de commits do [conventionalcommits](https://www.conventionalcommits.org/en/v1.0.0/)
Use o pacote NPM [git-commit-msg-linter](https://www.npmjs.com/package/git-commit-msg-linter)
